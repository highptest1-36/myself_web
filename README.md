[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/highptest1-36/myself_web)

**Steps:**
1. Click the "Deploy with Vercel" button above
2. Sign in with GitHub
3. Import the repository
4. Deploy automatically
5. Get your live URL: `yourname.vercel.app`

**Benefits:**
- ✅ Automatic deployments on push
- ✅ Free SSL certificate
- ✅ Global CDN
- ✅ Custom domain support
- ✅ Analytics included

#### Option 2: GitHub Pages

```bash
# 1. Install gh-pages
npm install --save-dev gh-pages

# 2. Add deploy script to package.json
"scripts": {
  "deploy": "next build && next export && gh-pages -d out"
}

# 3. Deploy
npm run deploy
```

Your site will be live at: `https://highptest1-36.github.io/myself_web`

#### Option 3: Netlify

1. Build the project: `npm run build`
2. Go to [Netlify](https://www.netlify.com/)
3. Drag & drop the `out/` folder
4. Get instant deployment with custom domain

## 🐛 Troubleshooting

### Port Already in Use

If port 3000 is already in use:

```bash
npm run dev -- -p 3001
```

### Images Not Loading

**Solution 1**: Check file paths
- Images in `public/` are referenced as `/image.jpg`
- No need for `/public/` prefix

**Solution 2**: Check Next.js config
```javascript
// next.config.mjs
images: {
  unoptimized: true,  // Required for static export
}
```

### Build Errors

Clear cache and reinstall:

```bash
rm -rf .next node_modules package-lock.json
npm install
npm run dev
```

### Dark Mode Not Working

Make sure `ThemeProvider` is in the root layout:

```typescript
// app/layout.tsx
<ThemeProvider attribute="class" defaultTheme="dark">
  {children}
</ThemeProvider>
```
